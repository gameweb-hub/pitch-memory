Kebijakan Privasi — Pitch Memory

Terakhir diperbarui: 17 Juli 2026

Pitch Memory ("aplikasi") dikembangkan oleh PT. ZUMNAA INTSYS INDONESIA. Halaman ini menjelaskan bagaimana aplikasi menangani data.

Data yang kami kumpulkan

Tidak ada. Aplikasi tidak mengumpulkan, menyimpan di server, atau membagikan data pribadi apa pun. Aplikasi tidak memerlukan koneksi internet, tidak menggunakan analitik, dan tidak menampilkan iklan.

Penyimpanan lokal

Progres permainan (level yang terbuka dan perolehan bintang) disimpan hanya di perangkat Anda. Data ini tidak pernah dikirim ke mana pun dan akan terhapus bila Anda menghapus aplikasi.

Izin

Aplikasi tidak meminta izin sensitif (lokasi, kontak, kamera, mikrofon, dsb.).

Anak-anak

Aplikasi aman untuk segala usia dan tidak mengumpulkan data dari siapa pun, termasuk anak-anak.

Perubahan

Kebijakan ini dapat diperbarui sewaktu-waktu. Perubahan akan dipublikasikan di halaman ini.
